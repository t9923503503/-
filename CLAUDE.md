# Памятка для Claude / ИИ-агентов

## Перед работой

1. Прочитай **`STATUS.md`** — актуальный этап, чеклист Фазы 5, зоны файлов, BLOCKED.
2. Прочитай **`DEVELOPMENT_PLAN.md`** — роли (ARCH / FORMAT / QA), протокол отметок задач (`🔄` / `✅`), зависимости.

## Текущий фокус

- **Фаза 5** — в работе: задачи **S5.1–S5.7** (ARCH) и **S5.8–S5.11** (FORMAT). Не начинай задачи из закрытых фаз, если не попросили явно.

## i18n — чтобы не путаться

- **Фаза 4 / A4.1** — уже сделана **инфраструктура**: `shared/i18n.js`, `locales/*.json`, интеграция в shared.
- **Экраны** (home, roster, навигация, KOTC и т.д.) переводятся в **Фазе 5** задачами **S5.8–S5.11**. Русский текст в `home.js` и подобных файлах — ожидаемо до завершения этих задач, это не «незакрытый A4.1».

## Синхронизация документов

После завершения задачи обновляй **`DEVELOPMENT_PLAN.md`** (строка задачи) и **`STATUS.md`** (чеклист + CHANGELOG), как описано в шапке плана.
