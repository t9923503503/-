# На удаление

Сюда перенесены файлы из корня репозитория, которые не нужны для сборки и разработки.

## Перед окончательным удалением

1. Просмотрите содержимое — что-то может пригодиться локально (скриншоты, старые заметки).
2. **`DEPLOY_lpvolley_root.md`** — содержит **конкретный IP сервера и шаги SSH**. Не публикуйте и не коммитьте в открытый репозиторий; для продакшена храните доступы вне git (менеджер секретов, `.env` на сервере). Универсальный чеклист: [`docs/DEPLOY.md`](../docs/DEPLOY.md).
3. **`ARCHITECTURE_root_legacy.md`** — старый обзор архитектуры от 2026-03-23; актуальные точки входа: [`README.md`](../README.md), [`docs/ARCHITECTURE_PHASES_6_8.md`](../docs/ARCHITECTURE_PHASES_6_8.md).

После проверки папку можно удалить целиком:

```powershell
Remove-Item -Recurse -Force to-delete
```
